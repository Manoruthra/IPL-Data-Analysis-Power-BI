# IPL-Data-Analysis-Power-BI
# Project Overview:
The aim of this project is to analyze Indian Premier League (IPL) data using Power BI to generate meaningful insights about teams, players, matches, toss decisions, bowling performance and batting statistics and fielding performance.

# Objective:
•	Identify the most successful IPL teams and players.

•	Analyze batting and bowling performance.

•	Toss decisions and their impact on match outcomes.

•	Find top performing fielders.

•	Analyze city and stadium wise match statistics.

•	Generate insights useful for team strategy.

# Data Cleaning and Preparation:
•	Handled null values in city column using venue information.

•	Remove umpire3 column because it contained 100% null values.

•	Corrected Teams name using replace values.

•	Created a relationship between matches and deliveries tables.

# KPIs Created:
•	Total Matches

•	Total Runs

•	Total Wickets

•	Total Sixes

•	Total Fours

•	Orange Cao Holder

•	Most Successful Team

•	Most player of the Match Awards

# Dashboard Visualizations:
# Page 1: Summary Page

•	KPI Cards

•	Winning Stadiums

•	Toss Decision Analysis

•	Top Winning Team

# Page 2: Overview Page
•	Top Bowlers and Bottom Bowlers

•	Top Fielders

•	Fielding Statistics Table

# Key Insights Generated:
•	Royal Challengers Bangalore emerged as one of the most successful IPL Teams.

•	CH Gayle received the highest player of the match awards.

•	Teams Choosing to field after winning the toss had higher win percentages.

•	SL malinga was among the top wicket taking bowlers.

# Tools and Technology Used:

•	Power BI

•	Power Query

•	DAX

•	CSV datasets
# Conclusion:
The IPL data analysis Dashboard successfully transformed raw IPL datasets into meaningful visual insights using Power BI. The project demonstrated skills in data cleaning, data modeling, DAX Calculations and dashboard designs.
